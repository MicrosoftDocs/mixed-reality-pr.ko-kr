# <a name="tooltip"></a>Tooltip

![Tooltip](images/UX_Hero_Tooltip.jpg)

도구 설명은 다른 컨트롤이나 개체에 연결된 간단한 설명입니다. 이러한 정보는 더 가까운 검사 시 힌트 또는 추가 정보를 전달 하는 데 사용 됩니다. 도구 설명을 통해 사용자는 실제 환경에서 직접적으로 설명 되지 않은 잘 모르는 개체를 이해할 수 있습니다.

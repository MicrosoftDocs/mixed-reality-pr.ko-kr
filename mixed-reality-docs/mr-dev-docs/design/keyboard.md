# <a name="keyboard"></a>키보드

![키보드](images/UX_Hero_Keyboard.jpg)

혼합 현실 애플리케이션은 대상 플랫폼의 기능에 따라 동작하는 시스템 키보드를 언제든지 호출할 수 있습니다. 예를 들어 HoloLens 2 키보드는 직접 손 상호 작용을 지원하지만 HoloLens(1세대) 키보드는 응시 및 에어 탭을 지원합니다.

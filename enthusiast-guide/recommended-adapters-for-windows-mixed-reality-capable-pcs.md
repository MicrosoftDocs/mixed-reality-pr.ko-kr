# <a name="recommended-adapters"></a>권장 어댑터
> [!NOTE]
> Microsoft는 이러한 Windows Mixed Reality PC 어댑터 권장 사항을 업데이트 하 고 추가 하 고 수정 하 고 있습니다. 최신 버전을 정기적으로 다시 확인 하세요.

[Windows Mixed Reality 호환 PC](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) 에는 몰입 형 헤드셋을 연결 하는 데 필요한 전체 크기의 HDMI 및 USB 3.0 포트가 없을 수 있습니다. 이 경우 어댑터가 필요 합니다. 또한 일부 어댑터는 PC 또는 헤드셋 케이블에 연결 하기 위한 결합기를 추가 해야 합니다. 사용 가능한 다양 한 어댑터 유형과 특정 모델에 대 한 몇 가지 권장 사항을 나열 하 여 시작 합니다.

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Windows Mixed Reality 호환 USB Bluetooth 어댑터

헤드셋이 2018 이후에 릴리스된 경우 어댑터와 컨트롤러가 헤드셋에 쌍으로 연결 되지 않아도 됩니다.

2017 또는 이전 버전에서 제공 되는 장치에는 컨트롤러용 어댑터가 필요 합니다. PC에 [Windows Mixed Reality 동작 컨트롤러](controllers-in-wmr.md#pair-motion-controllers)를 페어링 하기 위한 Bluetooth가 포함 되어 있지 않은 경우 플러그형 USB Bluetooth 4.0 저 에너지 마이크로 어댑터와 같은 어댑터를 사용할 수 있습니다.

> [!NOTE]
> Microsoft는 모든 어댑터의 기능, 안정성 또는 품질을 제조, 테스트, 인증 또는 보장 하지는 않습니다. 모든 문제에 대 한 질문, 문제 또는 문제는 어댑터의 판매자 또는 제조업체에 문의 해야 합니다.

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Windows Mixed Reality 헤드셋에서 외부 USB 3.0 허브 사용

PC에 열려 있는 USB 포트가 충분 하지 않은 경우 외부 USB 3.0 허브에 Windows Mixed Reality 헤드셋을 사용할 수 있습니다. 외부 USB 3.0 허브는 **AC 어댑터를 통해 외부에서 전원이 공급 되어야** 헤드셋에 충분 한 전원을 제공할 수 있습니다.

또한 외부 USB 3.0 허브 사용에 대 한 다음 지침을 참조 하세요. 
* USB 허브는 USB 3.0 허브 여야 합니다.
* USB 3.0 허브는 PC의 USB 3.0 포트에 연결 되어 있어야 합니다.  
* USB 3.0 허브는 외부에서 전원이 공급 되어야 합니다 (AC 어댑터가 추가 기능을 제공 하는 경우).
* 48 W 전원 어댑터를 사용 하는 플러그형 10 포트 USB 3.0 SuperSpeed 허브는 Windows Mixed Reality 헤드셋 및 대부분의 Pc에서 잘 작동 하는 것으로 알려져 있습니다.

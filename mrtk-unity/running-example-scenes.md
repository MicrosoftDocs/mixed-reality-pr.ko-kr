# <a name="example-scenes"></a>예제 장면

MRTK는 공간 사용자 환경에 대 한 MRTK의 기능 및 구성 요소를 보여 주는 다양 한 유형의 예제 장면을 제공 합니다. 개로 나누어서 예제 장면을 통해 기능을 이해 하 고 프로젝트에 적용 하는 데 도움이 될 수 있습니다. 

## <a name="how-to-acquire-example-scenes"></a>예제 장면을 얻는 방법

### <a name="using-mixed-reality-feature-tool-and-unity-package-manager"></a>혼합 현실 기능 도구 및 Unity 패키지 관리자 사용

혼합 현실 [기능 도구](/windows/mixed-reality/develop/unity/welcome-to-mr-feature-tool) 를 통해 **혼합 현실 Toolkit 예제** 패키지를 다운로드 하 여 가져올 수 있습니다.

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_MRFT.png" width="550" alt="Example Package 1"><br/>

Unity에서 메뉴 창을 사용 하 여 **Project > 사용자 지정에서 > 패키지 관리자 >** 하 고 **혼합 현실 Toolkit 예** 를 선택 합니다. 

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_2.png" width="300" alt="Example Package 2"><br/>

패널의 오른쪽에 있는 목록에서 예제 장면 이름 옆에 있는 **Project로 가져오기** 단추를 클릭 합니다.  예를 들어, **데모-수동 추적** 옆의 **Project 단추로 가져오기** 를 클릭할 수 있습니다. 

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_3.png" width="650" alt="Example Package 3"><br/>

가져온 후에는 **자산 > 샘플** 폴더 아래에서 찾을 수 있습니다.
**HandInteractionExamples** 장면은 MRTK의 공간 상호 작용 및 UI 구성 요소가 발생 하기 시작 하는 좋은 장소입니다.

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_4.png" width="650" alt="Example Package 4"><br/>

### <a name="directly-downloading-and-importing-packages-from-github"></a>GitHub에서 패키지 직접 다운로드 및 가져오기

Mixed Reality 기능 도구를 사용 하지 않는 경우 Toolkit MixedReality를 직접 다운로드 하 여 가져올 수 있습니다. **unitypackage의 예.** [mrtk GitHub의 릴리스 페이지](https://github.com/microsoft/MixedRealityToolkit-Unity/releases)

**> > 패키지 가져오기 패키지** 를 사용 하 여 unitypackage를 가져옵니다. 가져온 후에는 **자산 > MRTK > 예제 > 데모** 에서 예제 장면을 찾을 수 있습니다.
